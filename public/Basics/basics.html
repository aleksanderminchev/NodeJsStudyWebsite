Console, Variables, Constants

You can create either constants or variables with 'const' and 'let' keyword. Using them like in the picture initiliazes the constants/variables.
The const declaration creates a read-only reference to a value. It does not mean the value it holds is immutableâ€”just that the variable identifier cannot be reassigned. For instance, in the case where the content is an object, this means the object's contents (e.g., its properties) can be altered.


Console.log() is the function which prints to the console in NodeJs.
You can concatinate variables you would like to print out with the '+' sign. Another way is using ${expression} which is template literals(strings). The expression inside gets concatinated to the other string.

Numbers

ParseInt or ParseFloat methods allow us to convert a string to a int even if it has non-number characters. '+' before the const/variable also servers a simillar function in printing to console as it denotes that the following variable is a number. Using Number() on any string converts it to a number but if non-number characters are present it gives back NaN.
toFixed() function allows us to set how many digits to show after the decimal point returning a string.


Strings

Strings in NodeJs can also function as an array of characters. We can either use search() or chartAt() functions to find the character we want. Another way is to use an array syntax to lookup a specific position in the array and that would be the following syntax - array[i].
