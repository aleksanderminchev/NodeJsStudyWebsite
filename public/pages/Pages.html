HTML Pages

By using the res.sendFile() we are able to send entire HTML pages instead of just specific HTML code like before. We need to establish our folder from where we can get these pages and design them. Normally that would be the public folder with subfolder for every html page. We can style these pages however we like with our own custom css and js scripts to run on them. By using __dirname we can make it so this code is runnable everywhere in any machine as long as they have the source code and do not change the file hierarchy. 


    app.use(express.static('public'));

    app.get("/", (req, res) => {
      res.sendFile(__dirname + "/public/welcome/welcome.html");
    });
    app.get("/dragons", (req, res) => {
      res.sendFile(__dirname + "/public/dragons/dragons.html");
    });
